What is readme?


Readme is a platform where anyone can read any book, at anytime.

Students nowadays have to borrow books from libraries and the book is issued only for a limited time, and we know how hard it is.

So, our team came up with a solution and decided to bring all the books you want on your nearest internet compatible devices, So now you don't have to worry about returning the books and finally you can say "Goodbyes" to the library.

Currently Readme is in a closed beta stage and we are planning to start a public beta very soon.
